# blurrcat's dotfiles

0. get this repo: `git clone https://github.com/blurrcat/dotfiles.git ~/.dotfiles && cd ~/.dotfiles`
1. `./bootstrap/brew.sh` or `./bootstrap/apt.sh` depending on your OS
2. profit!
